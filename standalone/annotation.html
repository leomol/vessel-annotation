Outlining instructions

  • Outline the edge of the big blob at the center of each image (including the first image, which you can use as an example).
  • If half of the edge is blurry, assume a circular or an elliptical shape.
  • Don't include bright extrusions or bright dots right next to the main blob.
  • If there is a second crescent shape around the blob, then you can include it if laying on top of the blob but don't include it if there is space between the blob and the crescent shape.
  • Even if the system accepts your submission, approval of your work will be decided later (2+ weeks) by a human, so please read and follow the instructions carefully so that your and our efforts are use effectively.
  • Note for reviewers: this is a standalone version of the software for demonstration purposes only which prompts to load images and downloads the results after completing the task. The crowd-source version presents the user with those images and stores data automatically in the server.

Controls

  • This task requires using a computer mouse with a scroll button.
  • Click on Browse to select a folder of images.
  • To change to another image use the left/right arrow keys (faster) or click on Previous/Next.
  • To move individual markers, click and drag each individual marker.
  • To push markers with the circle tool, click in an empty area and drag the mouse.
  • To move markers enclosed by the circle tool, shift+click and drag the mouse.
  • To change the tool size, scroll down/up.
  • To reset the marker arrangement, right-click on the image.
  • The 50s video tutorial walks you thru these steps.
